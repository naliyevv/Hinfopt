HINFOPT v1.0

This is a MATLAB package for optimizing the H-infinity norm of a 
large-scale rational transfer function via projection. The main routine
is hinfopt.m

References:

[1] N. Aliyev, P. Benner, E. Mengi, and M. Voigt.
    A subspace framework for H-infinity norm minimization. Submitted for
    publication, 2019.
